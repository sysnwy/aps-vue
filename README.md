# Our Stock
This template helps you get started with Vue3 development on Vite using Finnhub API to display real-time stock/crypto data in the `US` region and display market news. This allows users to easily access market data.

<img src="https://raw.githubusercontent.com/sysnwy/aps-vue/refs/heads/main/public/icon.png" alt="brand" style="border-radius:50px; width:80px; height:80px;" />

Further configuration for Vite server
See https://vite.dev/config/

Installation Project Preparation
```bash
npm install
```

Get Finnhub API Key
1. Go to https://finnhub.io/ and create an account to get your first API.
2. Get the API key from the dashboard.

Run the application 
```bash
npm run dev
```

Compilation and Minification for Production
```bash
npm run build
```
